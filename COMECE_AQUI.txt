🚀 PARA GERAR O APK:
═══════════════════════════════════════════════════════════════

PASSO 1: Gerar Ícones (OBRIGATÓRIO)
───────────────────────────────────
  $ python gerar_icones.py

  ✨ Cria:
     • data/icon.png (192x192)
     • data/presplash.png (1280x720)


PASSO 2: Instalar Dependências
───────────────────────────────
  $ pip install --upgrade buildozer cython


PASSO 3: Compilar APK
─────────────────────
  Automático (Recomendado):
  $ python preparar_apk.py

  Ou Manual:
  $ buildozer android debug

  ⏳ Tempo: 30-60 minutos primeira vez
  📦 Resultado: bin/controlevendas-1.0.0-debug.apk
